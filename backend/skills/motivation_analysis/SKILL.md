# 动机分析技能

当用户询问动机或需要分析嫌疑人动机时,请按以下步骤操作:
1. 如果用户提及具体嫌疑人姓名,使用 `search_notes` 搜索包含该姓名的便签,关键词可以是嫌疑人名字或相关描述
2. 使用 `get_timeline` 获取所有时间线信息,特别关注与嫌疑人有交集的事件.
3. 综合分析,列出:
   - 可能的直接动机 (如利益,仇恨,情感纠葛)
   - 可能的深层动机 (如恐惧,控制,保护他人)
4. 如果信息不足,明确说明"目前还无法确定动机"

输出格式示例：
【动机分析】
- 直接动机：...
- 深层动机：...
